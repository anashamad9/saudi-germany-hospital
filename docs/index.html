What this API returns

One JSON file containing every brand in the SGH partnership sheet. Each brand entry includes its name, the geo markets it targets, the discount terms, the active coupon code, and the brand logo URL.

The file is regenerated every 30 minutes by a GitHub Actions job that reads the source Google Sheet directly. The last_updated field in the response tells you exactly when the last sync ran.

Connect from a website (JavaScript)

1

Fetch the JSON in your page

Call the endpoint with the browser Fetch API. No API key or header is needed.

fetch("https://anashamad9.github.io/saudi-germany-hospital/api/brands.json")
  .then(res => res.json())
  .then(data => {
    const brands = data.brands;
    brands.forEach(brand => {
      console.log(brand.brand_name, brand.coupon_code);
    });
  });
2

Filter by geo

Use the targeting_geos array to show only brands relevant to the user's region.

const ksaBrands = data.brands.filter(b =>
  b.targeting_geos.some(g => g.includes("KSA"))
);
3

Render the data

Loop over the filtered brands and build your UI elements from the fields.

ksaBrands.forEach(brand => {
  const card = document.createElement("div");
  card.innerHTML = `
    <img src="${brand.brand_logo}" alt="${brand.brand_name}" />
    <h3>${brand.brand_name}</h3>
    <p>${brand.discount}</p>
    <code>${brand.coupon_code}</code>
  `;
  document.getElementById("brands-container").appendChild(card);
});

Connect from a mobile app (React Native)

import { useEffect, useState } from "react";

export function useBrands() {
  const [brands, setBrands] = useState([]);
  useEffect(() => {
    fetch("https://anashamad9.github.io/saudi-germany-hospital/api/brands.json")
      .then(r => r.json())
      .then(d => setBrands(d.brands));
  }, []);
  return brands;
}
Tip: Cache the response locally (AsyncStorage or similar) and only refetch when last_updated is newer than your cached timestamp. This avoids unnecessary network calls on every app launch.

Connect from Python (backend or script)

import requests

API = "https://anashamad9.github.io/saudi-germany-hospital/api/brands.json"

Connect from a no-code tool (Zapier / Make / n8n)

1

Add an HTTP Request node

In Zapier use the "Webhooks by Zapier" action. In Make use the "HTTP - Make a request" module. In n8n use the "HTTP Request" node.

2

Configure the request

Method: GET

URL: https://anashamad9.github.io/saudi-germany-hospital/api/brands.json

No headers or authentication needed.

3

Map the fields

The response body is parsed automatically. Map brands[0].brand_name, brands[0].coupon_code, etc. to downstream steps in your workflow.

CORS

GitHub Pages serves all files with permissive CORS headers (Access-Control-Allow-Origin: *). You can call this API from any browser, app, or server without proxy workarounds.

Top-level fields

last_updatedstringISO 8601 UTC datetime of the last successful sync from Google Sheets (e.g. "2026-06-09T10:00:00Z")
sourcestringIdentifies the data origin. Always "Digizag X SGH Brands"
totalnumberTotal count of brand objects in the brands array
brandsarrayList of brand objects (see below)

Brand object fields

brand_namestringDisplay name of the brand
targeting_geosstring[]Array of geo markets the brand targets. Common values: "KSA", "UAE", "GCC", "KSA/UAE". A brand can target multiple geos.
discountstringShort summary of the discount offer. May contain newlines for multi-tier discounts.
discount_contentstringFull discount details and terms as entered in the sheet.
websitestringThe brand's website URL.
coupon_codestringActive promo or coupon code for this brand. Apply at checkout on the brand's website.
brand_logostringFull URL to the brand logo image. Suitable for use in an img src attribute directly.

Notes

Fields will never be missing from a brand object but may be empty strings if the corresponding cell in the sheet was blank.

The targeting_geos field is always an array even when a brand targets only one region.

Newline characters (\n) in discount and discount_content are intentional and represent line breaks from the original sheet cell.
